## Categories

- **texture** — repeated surface pattern (stripes, spots, scales, fur)
- **color** — dominant hue with little other structure
- **part** — specific object part (beak, wing, leg, eye)
- **scene** — background or environment (water, sky, grass)
- **semantic** — high-level class concept (bird, animal)
- **unclear** — no consistent theme across top patches
